# Template Sistem Login & Register (Beta)

Project ini adalah template sederhana untuk sistem autentikasi mahasiswa menggunakan PHP Native dan MySQL.
Project ini dibuat untuk keperluan tugas kuliah dan masih dalam tahap pengembangan (Work in Progress).

## Fitur Saat Ini
- Login User
- Registrasi User Baru
- Koneksi Database MySQL (Lihat `config.php`)

## Cara Instalasi
1. Download atau Clone repositori ini.
2. Import file `database.sql` ke database lokal Anda.
3. Sesuaikan konfigurasi database di file `config.php`.
4. Jalankan di server lokal (XAMPP/Laragon).
